Colophon

The tools and services used to create this website.

The code for this website is hosted on GitHub and created using Visual Studio Code. Automated deployments are done by GitHub Pages. Eleventy is used as the static site generator and the theme is a custom one created by me.

The font in use is Instrument Sans. Syntax highlighting is done using the 11ty Syntax Highlighting plugin and the theme is Catppuccin Mocha.

Tracking is done using Umami. I self-host the tracking script at /scripts/script.js and all data gets sent to their EU based datacenters. No identifiable information is collected.
