The code for this website is on [GitHub](https://github.com/mijndert/blog) and created using [Visual Studio Code](https://code.visualstudio.com/). Automated deployments are done by [CloudFlare Pages](https://pages.cloudflare.com/). [Hugo](https://gohugo.io/) is used as the static site generator and the theme is [Rocinante](https://github.com/mavidser/hugo-rocinante).
